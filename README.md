# Docker compose file for elasticsearch and kibana

## To start

First download and install docker for mac which includes docker-compose, then

```bash
  > cd into the elasticsearch folder
  > docker-compose up -d
```

## To stop

```bash
  > cd into the elasticsearch folder
  > docker-compose down
```

### Elasticsearch
Access Kibana at "http://localhost:5601"

### Kibana
Access Kibana at "http://localhost:5601"
